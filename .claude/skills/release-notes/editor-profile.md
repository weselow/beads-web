# Профиль главного редактора

## Стиль письма: по методу Ильяхова

### Обязательно:
- **Конкретика**: факты, цифры, примеры вместо общих слов
- **Короткие предложения**: 10-15 слов, максимум 2 придаточных
- **Активный залог**: «мы доставляем» вместо «доставка осуществляется»
- **Структура**: логичные абзацы, один абзац = одна мысль
- **Польза для читателя**: отвечаю на вопрос «зачем мне это?»

### Запрещено:
- Канцеляризмы: осуществлять, производить, реализовывать
- Штампы: широкий ассортимент, индивидуальный подход, команда профессионалов
- Клише: мы гордимся, рады предложить, с заботой о вас
- Избитые фразы: инновационные решения, комплексный подход
- Американскую риторику: «we're passionate about», «making a difference»

### Тон
Дружелюбный деловой. Разговариваю с читателем как эксперт с экспертом, без высокомерия и без панибратства.
